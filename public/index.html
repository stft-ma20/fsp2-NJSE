Basic Express Server!

/Toys

acces and modify database of toys.

toy object properties : id (string, automatic), name(string), price (number), category(string), info(string),imga_url (string), cdate_created (date, optional, default: date().nwo), user id (automatic)

***admin:
boss@org.com, 123123

# First path description quary parameters
1 GET /toys Get all toys from databae.

?page : get items in pges (10 items per page, page >=1)
2 GET /toys/search Find and get realevant toys using a keyword (searching in item's name & info).

?s : value of search keyword

?page : get items in pges (10 items per page, page >=1)
3 GET /toys/category/ Get items from the realevant category.

:cat : value of category to search

?page : get items in pges (10 items per page, page >=1)
3 GET /toys/prices/ Get items in the realevant price range.

?min : min price of items

?max : max price of items
3 POST /toys Add new item to database

* must provide valid user token in request's header.

* must provide valid new toy object in the request payload.
3 DEL /toys/:delID Delete item from database.

* must provide valid user token in request's header.

* must provide valid new toy object in the request payload.

:delID : _id of item to delete
3 PUT /toys/:editID Edit item from database

* must provide valid user token in request's header.

* must provide valid new toy object in the request payload.

:editID : _id of item to edit

/users

acces and modify database of users.

toy object properties id (string, automatic), name (string), email (string), password (string), role (optional, USER or ADMIN, default: USER), creation date (automatic).

# First path description
1 Get /users See all users.
1 Post /users Add new user to database.

* must provide valid new user object in the request body.
2 POST /users/login Login as user.

* must provide valid login details in the request body.
